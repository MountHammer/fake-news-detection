# 📰 Fake News Detection

## 📌 Description
This project uses Machine Learning to classify news headlines as **REAL or FAKE** using Natural Language Processing techniques.

---

## 🚀 Features
- Takes user input (news headline)
- Predicts whether the news is real or fake
- Uses TF-IDF for text processing
- Uses Logistic Regression model

---

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn

---

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/MountHammer/fake-news-detection.git

2. Install dependencies:
   pip install pandas scikit-learn

3. Run the program:
   python main.py

---

## 📊 Example

Input:
Breaking news: market crashes

Output:
This news is REAL ✅
